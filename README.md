# 数据爬取
步骤:  
1. 网站界面调研(浏览器地址栏的统一资源定位符变化、f12等)  
2. 模拟用户请求  
3. 数据解析（BeautifulSoup+正则）  
4. 数据存储  
5. 分析和可视化  

  
较为常见的两种网站数据：  
1. 前后端分离的 api数据爬取
2. 基于页面渲染的 xpath数据爬取
other：大多带有比较难搞的反爬虫机制。
